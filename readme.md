# Test-Adeo

## Prerequisites

- NodeJS (latest)
- Yarn (latest)

## Get

`git clone https://github.com/qjacquet/test-adeo.git`

`cd test-adeo`

## Install

`npm install`

## Test

Tests with jest

`npm test`

## Run

`node app.js --filter="ry"`

`node app.js --count`
